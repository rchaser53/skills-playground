# Website Summary Input Example

以下は、サイト要約skillに渡すサンプル入力です。

## 目的

- AI画像生成サービス候補をざっくり比較したい
- 価格、主な用途、特徴的な機能を短く把握したい
- 初見ユーザー向けにわかりやすく要約してほしい

## 対象サイト

1. OpenAI
https://openai.com/
メモ: 画像生成まわりの製品導線や、何ができるのかを中心に見たい。

2. Stability AI
https://stability.ai/
メモ: 生成モデルの種類、開発者向けか一般向けかを確認したい。

3. Midjourney
https://www.midjourney.com/
メモ: クリエイティブ用途での強みがわかるとよい。

4. Markdown link example
[Adobe Firefly](https://www.adobe.com/products/firefly.html)
メモ: Adobe製品との連携や商用利用まわりが気になる。

## 出力してほしい形式

- 各サイトごとに3-5点で要約
- 最後に、違いがわかる比較メモを付ける
- 取得できなかったURLがあれば明記する
